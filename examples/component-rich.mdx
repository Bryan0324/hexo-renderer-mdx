# Component-Rich Blog Post

This example demonstrates how to create and use custom React components in your MDX content.

## Using Custom Components

### Info Boxes

<InfoBox type="info">
  This is an informational message with custom styling!
</InfoBox>

<InfoBox type="warning">
  Be careful! This is a warning message.
</InfoBox>

<InfoBox type="success">
  Great job! Your operation was successful.
</InfoBox>

<InfoBox type="danger">
  Error! Something went wrong.
</InfoBox>

## Mixing Markdown and JSX

You can seamlessly mix markdown and JSX:

Regular markdown **bold** and *italic* text, followed by a custom component:

<InfoBox type="success">
  Inside this box, you can use **markdown** _formatting_!
  
  - List item 1
  - List item 2
</InfoBox>

## Code Examples

Here's how to define a simple component in MDX:

```jsx
export const Button = ({ children, color }) => (
  <button style={{ backgroundColor: color }}>
    {children}
  </button>
);
```

Then use it like this:

```jsx
<Button color="blue">Click Me</Button>
```
