# Welcome to MDX!

This is a simple blog post written in MDX. You can use all the familiar Markdown syntax:

## Text Formatting

- **Bold text**
- *Italic text*
- ~~Strikethrough~~
- `inline code`

## Code Blocks

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

## JSX Components

But you can also use JSX! Here's a styled alert box:

<div style={{
  padding: '15px',
  backgroundColor: '#fff3cd',
  border: '1px solid #ffc107',
  borderRadius: '5px',
  marginTop: '20px',
  marginBottom: '20px'
}}>
  ⚠️ <strong>Note:</strong> This is a custom alert box created with JSX!
</div>

## Interactive Elements

Create custom buttons:

<button style={{
  backgroundColor: '#007bff',
  color: 'white',
  padding: '12px 24px',
  border: 'none',
  borderRadius: '5px',
  fontSize: '16px',
  cursor: 'pointer'
}}>
  Click Me!
</button>

## Conclusion

MDX gives you the power of React components with the simplicity of Markdown!
